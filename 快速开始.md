# 🍅 番茄待办提醒 - 3分钟快速开始

## 一键启动（推荐）

如果你的系统已经配置好Node.js和MySQL，直接运行：

```bash
./start.sh
```

启动脚本会自动：
- ✅ 检查环境
- ✅ 安装依赖（首次运行）
- ✅ 初始化数据库（首次运行）
- ✅ 启动应用

## 手动启动

### 1️⃣ 安装依赖

```bash
npm install
```

### 2️⃣ 初始化数据库

确保MySQL正在运行（root/1234），然后执行：

```bash
npm run db:init
```

### 3️⃣ 启动应用

```bash
npm start
```

等待几秒钟，应用窗口会自动打开！

## 🆘 遇到问题？

### MySQL连接失败？

1. 检查MySQL是否启动
2. 确认账号密码：root/1234
3. 修改配置文件：`src/database/config.js`

### 端口被占用？

```bash
# 结束占用3000端口的进程
lsof -i :3000
kill -9 [PID]
```

### 依赖安装失败？

```bash
# 清除缓存重新安装
npm cache clean --force
rm -rf node_modules
npm install
```
