## 🆘 遇到问题？

### MySQL连接失败
- 检查MySQL是否启动：`mysql -u root -p1234 -e "SELECT 1"`
- 修改配置文件：`src/database/config.js`

### 端口被占用
```bash
lsof -i :3000
kill -9 [PID]
```

### 依赖安装失败
```bash
npm cache clean --force
rm -rf node_modules
npm install
```

更多问题请查看 `SETUP.md` 中的故障排除章节。
